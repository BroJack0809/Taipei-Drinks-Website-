# 台北手搖飲菜單 🧋

一個精美的台北手搖飲料店菜單網站,提供完整的飲品資訊、分類篩選和中英文雙語支援。

[![LIVE](https://img.shields.io/badge/LIVE-GitHub%20Pages-success?style=for-the-badge)](https://brojack0809.github.io/Taipei-Drinks-Website-/)
[![DEMO](https://img.shields.io/badge/DEMO-View%20Site-brightgreen?style=for-the-badge)](https://brojack0809.github.io/Taipei-Drinks-Website-/)

## ✨ 功能特色

- 🏪 **5家熱門飲料店** - COMEBUY、可不可熟成紅茶、五桐號、50嵐、一沐日
- 📋 **完整菜單** - 超過 300 種飲品,包含詳細分類和價格
- 🔍 **智慧搜尋** - 快速搜尋店家、飲品名稱或口味
- 🏷️ **分類篩選** - 咀嚼系、凍飲控、鮮奶歐蕾、清爽果茶
- 🌐 **雙語支援** - 繁體中文 / English 即時切換
- 📱 **響應式設計** - 完美支援手機、平板、電腦
- 🎨 **現代化 UI** - 玻璃擬態設計、流暢動畫效果

## 🚀 快速開始

### 線上瀏覽
直接訪問: [https://brojack0809.github.io/Taipei-Drinks-Website-/](https://brojack0809.github.io/Taipei-Drinks-Website-/)

### 本地運行

```bash
# 克隆專案
git clone https://github.com/BroJack0809/Taipei-Drinks-Website-.git

# 進入專案目錄
cd Taipei-Drinks-Website-

# 使用任何靜態伺服器運行,例如:
# 使用 Python
python -m http.server 8000

# 或使用 Node.js
npx serve

# 然後在瀏覽器開啟 http://localhost:8000
```
